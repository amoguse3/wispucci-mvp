# Barem Guide — how points are actually awarded (clasa 9)

> The exam rewards *what you write for the grade*, not just knowing the answer. This guide encodes the grading logic so the tutor can coach students to **score**, not merely to understand. Pulled from official ANCE baremes (math sb25, history sb24, română sb25).

## Why this matters for the product
Most competitors teach content. Almost none teach **barem literacy** — the difference between a 1p and a 2p answer. That's a concrete, teachable edge. Every lesson should end with "here's how this is scored on the exam."

## Mathematics
- Partial credit **per step**. A worked item is scored: identify data → set up formula/equation → solve → final answer with unit.
- Example (sb25 item 8, linear system word problem, 5p): setting up the 2 equations = 2p (1p each), solving = 2p, correct final answer = 1p.
- Wrong final answer **with correct steps still earns points**.
- Any valid method reaching the answer = full marks. Graders don't require shown work unless the item asks for *rezolvare*.
- Fill-in-box items (casete): points only for the correctly filled box, usually 1p each.
- **Teach:** always write the steps even if unsure of the final number; never leave a worked item blank.

## History
- Universal pattern: **0p** missing/wrong · **1p** partial argument (no example/evidence) · **2p** full argument (reasoning + concrete example from a source or prior knowledge).
- Source-analysis items = a task verb (Numește/Identifică/Formulează/Apreciază, 1p) + Argumentează (2p).
- Essay (Subiectul III, 15p): sources integrated (2p) + structure intro/body/conclusion (3p, 1p each) + ≥3 arguments (7p: relevance 1p + 3 args ×2p) + references to personalities/chronology (2p) + scientific correctness (1p).
- **Teach the formula:** claim → reasoning → concrete example (a date, a personality, or a source quote). An argument without an example is capped at 1p.

## Limba română
- Competence-based: points for producing correct texts + precise metalanguage.
- Structured items score per requirement, e.g. opinion (item 5, 6p) = clear opinion (1p) + 3 valid arguments (3p) + coherence (1p) + length (1p).
- Rezumat (5p) = fidelity + essentials + no filler + grammatical coherence + exact word limit (1p each). **Count the words.**
- Syntactic functions: **specifying the complement type is mandatory** (complement direct / circumstanțial de timp); for subject/predicate/attribute it isn't.
- Punctuation items: identify the situation (1p) + state the exact rule (1p). Naming without the rule loses half.
- **Item 13 (5p language correctness)** spans the whole paper, descending by error count (5p impeccable → 0p if >8 errors), and is only granted if the productive items (1,2,3,5,6,10,11,12) were actually attempted.
- **Teach:** hit word limits exactly, always state the rule (not just the answer), and treat language correctness as a scored dimension throughout.
