# Pedagogy — How Wispucci Teaches

The lesson engine is built on evidence from the science of learning. Every step type maps to a proven principle.

## Core principles

### 1. Retrieval practice > rereading
Meta-analysis 2025 (Educational Psychology Review): pulling information out of memory beats rereading almost every time. Karpicke: testing itself changes memory, it doesn't just measure it.

**In the app:** every topic ends with quiz steps. Retrieval, not passive reading.

### 2. Pretesting (guess BEFORE you learn)
Memory & Cognition 2025: attempting an answer and being wrong before studying works BETTER than reading first. "Errorful learning" creates hooks the correct answer latches onto.

**In the app:** `pretest` step type opens some topics. Wrong answers are praised, never penalized, never count toward accuracy.

### 3. Desirable difficulties (Bjork)
If learning is too easy, you forget fast. Struggle helps. Tools: retrieval practice + spaced practice.

**In the app:** spaced review returns old topics, and returns them HARDER.

### 4. Cognitive Load Theory (Sweller)
Working memory holds ~4 elements at once, for seconds. Schemas build gradually.

**In the app:** one idea per screen. No walls of text. Short lessons plus return visits.

### 5. Worked examples + expertise reversal
- Novice: worked example first, then practice.
- Advanced: worked examples become harmful (redundancy effect), they need pure practice.
- Conclusion: the theory/practice ratio must SHIFT as the learner grows.

**In the app:** `worked` step + mode choice that lets advanced learners skip to practice.

### 6. Faded guidance
Golden sequence: worked example -> completion problem -> full problem solo. Scaffolding removed gradually.

**In the app:** step order `worked` -> `completion` -> `quiz`.

### 7. Interleaving example-problem pairs
Wiley 2020: alternating example-problem beats all-theory-then-practice OR only-practice. Some learners need theory first for control.

**In the app:** default interleaves; a mode toggle offers "theory first" or "dive into practice."

## Step types

| Type | Principle | What the user sees |
|------|-----------|--------------------|
| pretest | Errorful learning | Guess-first riddle, no penalty |
| concept | Cognitive load | One idea, clear formula |
| worked | Worked example effect | Fully solved example |
| completion | Faded guidance | Fill-in-the-blank problem |
| quiz | Retrieval practice | Solve solo, counts to accuracy |
| elaborate | Elaborative interrogation | "Why does this work?" then reveal |

## Tone

Mayer's personalization principle: conversational tone beats formal for retention and transfer. Wispucci speaks like a smart older friend who recently passed this exam. Warm, direct, slightly informal, encouraging on mistakes, living analogies. No memes for the sake of memes.
