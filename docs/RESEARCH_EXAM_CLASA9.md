## PART 1 — MATHEMATICS

### Exam structure
- 120 min, 12 items, mixed: fill-in boxes (casete) + full worked solutions (rezolvare desfășurată).
- Total 50 points. An **Anexă** (formula sheet) is always provided: `(a±b)²`, `(a−b)(a+b)=a²−b²`, area/volume formulas, `V=πr²h`, etc.
- **Barem rule (critical):** partial credit per step. From the 2025 barem, e.g. item 8 (linear system word problem): 2p for setting up the two equations (1p each), 2p for solving, 1p for the correct final answer. Wrong final answer + correct steps still scores. Any valid method that reaches the answer gets full marks; graders don't demand calculations unless the item asks.

### Recurring item types (2022-2025)
| Item slot | Topic | Seen in |
|---|---|---|
| 1 | Powers + integer/real arithmetic in fill-in boxes | every year (sb22-25, ss22-25) |
| 2 | Circle: central vs inscribed angle | sb24, sb25 |
| 3 | Function zeros (quadratic) | sb25 |
| 4 | Applied percentage problem | sb25 (televizor 5%), recurring |
| 5 | Radical simplification proving an integer | sb25 (√3(4−√12)−√48) |
| 6 | Quadratic equation, smallest real solution | sb25 (2x²−x−3=0) |
| 7 | Geometry: square/segment length with radicals | sb25 (5√5 cm) |
| 8 | Linear system disguised as word problem | sb25 (vitamins), recurring (mixtures, ages) |
| 9 | Inequalities from a function condition | sb25 (interval answer (1;4]) |
| 10 | Solid geometry volume comparison | sb25 (butter blocks → cylinder) |
| 11 | Algebraic expression + equation with condition of existence | sb25 |
| 12 | Quadratic function monotonicity | sb25 |

### Topic domains (ANCE program)

## PART 2 — HISTORY (Istoria românilor și universală)

### Exam structure (very stable 2023-2025)
- 120 min, total **50 points**, three subjects:
  - **Subiectul I (12 p):** 4 source-analysis items (image/map/stamp/poster). Each = *Numește/Identifică/Formulează/Apreciază* (1p) + *Argumentează* (2p).
  - **Subiectul al II-lea (23 p):** dossier of sources A-D (chronology + text + map). ~7 items: name a fact (1p), identify documents (2p), explain a term (2p), determine objectives with explanation (4p), describe consequences (4p), name a personality + 2 arguments (5p), express opinion + 2 arguments (5p).
  - **Subiectul al III-lea (15 p):** structured essay from sources A-C/D. Barem: use of sources (2p), coherent structure intro/body/conclusion (3p), ≥3 arguments (7p), references to personalities/chronology (2p), scientific correctness (1p).

### Barem logic (critical for the tutor)
Students lose points not from wrong facts but from **under-argumentation**. The pattern everywhere: 0p missing/wrong, 1p partial argument (no example/evidence), 2p full argument (reasoning + concrete example from source or prior knowledge). Teach the formula: **claim → reasoning → concrete example (date/personality/source)**. The essay explicitly requires intro+body+conclusion and ≥3 arguments each backed by a chronological reference or personality.

|---|---|---|---|

### Recurring exam themes to teach
Marea Unire 1918 · interwar period (crisis, totalitarianism, League of Nations) · WWII & territorial losses 1940 · Molotov-Ribbentrop · Cold War / Marshall Plan / interallied conferences · Sovietization of RSSM (collectivization, deportations, 1946-47 famine) · RM road to independence 1989-1991 · contemporary science/technology/culture · key personalities (Wilson, Clemenceau, Lloyd George, Stalin, Grigore Vieru). Terms frequently asked to *explain*: tratat, democrație, regim democratic.

## PART 3 — LIMBA ȘI LITERATURA ROMÂNĂ

### Exam structure
- 120 min. Single literary/artistic text (2025: Florina Ilis, *Cinci nori colorați pe cerul de răsărit*). 13 items. Competence-based, not memorization.
- **Item 13 = 5 points for language correctness** (orthography/punctuation/style), awarded in descending scale: 5p impeccable → 0p if >8 errors. Crucially it's only granted if items 1,2,3,5,6,10,11,12 are actually attempted.

### Item map (2025, representative)
| Item | Skill | Points | Barem note |
|---|---|---|---|
| 1 | Reading comprehension, full-sentence answers | 3 (1×3) | each answer a complete sentence |
| 2 | Define a figure of speech (comparația) + example | 2 (1+1) | definition + pertinent example |
| 3 | Summary (rezumat) 50-55 words | 5 (1×5) | fidelity, essentials, no detail, coherence, word limit |
| 4 | Lexicography: read a dictionary entry | 5 (1×5) | senses count, contextual sense, synonym, expression meaning, grammar info |
| 5 | Opinion + 3 arguments | 6 (1+3+1+1) | clear opinion, 3 valid args, coherence, length |
| 6 | Functional text: write an *anunț* | 4 | structure+data+length |
| 7 | Name 3 musical instruments (from text field) | 3 | |
| 8 | Apply an orthography rule with 2 examples | 2 | verb compound hyphenation |
| 9 | Syntactic functions of underlined words | 5 (1×5) | complement type must be specified |
| 10 | Justify comma use by the rule | 2 (1+1) | identify + state rule (subordonată concesivă) |
| 11 | Construct a 4-line dialogue | 4 | statute, coherence, logic, length |
| 12 | Descriptive text (book cover) | 4 | |
| 13 | Language correctness across the whole paper | 5 | descending by error count |

### Barem logic (critical)
Româna is **skill-based**: points come from *producing correct texts* (summary, opinion, dialogue, description, functional text) and *precise metalanguage* (figures of speech, syntactic functions, punctuation rules). Teach: how to write a rezumat to word limit, opinion structure (opinion + N arguments + coherence + length), functional-text formats (anunț/invitație/cerere), syntactic analysis, and comma/orthography rules. Language correctness is a standalone scored dimension across the whole paper.

### Recurring content areas
Reading comprehension · figures of speech (comparație, metaforă, epitet, personificare) · rezumat cu limită de cuvinte · lexicography/dictionary entries · argumentative opinion · functional texts (anunț, invitație, cerere, mesaj) · syntactic functions · punctuation rules (virgula, subordonate) · orthography rules · dialogue construction · descriptive/narrative composition.
